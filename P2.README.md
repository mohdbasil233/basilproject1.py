Employee Class – Property, Setter, Deleter Example
This Python program demonstrates how to use @property, setter, and deleter decorators in an object‑oriented design.

 Features
Employee Class with fname and lname attributes.

Property (@property): Generates an email dynamically from the employee’s first and last name (fname.lname@gmail.com).

Setter (@email.setter): Allows updating the employee’s name by assigning a new email string.

Deleter (@email.deleter): Resets the employee’s name to None when the email is deleted.

Explain Method: Returns a readable description of the employee.
